Button groups

Bootstrap allows you to group a series of buttons together (in a single line) in a button group

Use a div element with a class .btn-group to create a button group

Instead of applying button sizes to each button in a group use class .btn-group-lg|sm|xs to size all buttons in the group

Vertical button groups

Bootstrap also supports vertical button groups

Use the class .btn-group-vertical to create a vertical button group

Justified button groups

To span the entire width of the screen use the .btn-groupe-justified element
