Basic list groups

To create a list group use <ul> element with class .list-group, and <li> elements with class .list-group-item

You can also add badges to a list group. The badges will automatically be positioned on the right

Contextual classes

Contxtual classes can be used to color list items

The classes for coloring list items are:

  • .list-group-item-success
  • .list-group-item-info
  • .list-group-item-warning
  • .list-group-item-danger
