Toggleable/Dynamic tabs

To make the tabs toggleable add data-toggle="tab" attribute to each link

Then add a .tab-pane class with a unique ID for every tab and wrap them inside a <div> element with class .tab-content

If you want the tabs are fade in and out when clicking on them, add the .fade class to .tab-pane
