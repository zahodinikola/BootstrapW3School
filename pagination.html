Basic pagination

If you have the web site with lots of pages, you may wish to add some sort of pagination to each page

To create a basic pagination you can add a .pagination class to <ul> element

The pagination class provides pagination links

Pagination blocks can also be sized to a larger size or a smaller size

Add class .pagination-lg for larger blocks or .pagination-sm for smaller blocks
