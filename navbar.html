Bootstrap navigation bar


Navigation bars

A navigation bar is a navigation header that is placed on the top of the page

With Bootstrap a navigation bar can extend or collapse, depending on screen size.

A standart navigation bar is created with <nav class="navbar navbar-default">

Navbar forms

To add form elements inside the navbar, add the .navbar-form class to a form element and add an inputs

If there is more than one input, you should wrap it by div with a .form-group class
