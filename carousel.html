The Carousel plugin

Plugin can be included idividually (use Bootstrap's individual carousel.js file) or all at once (bootstrap.js or bootstrap.min.js)

How to create a basic carousel

Previous Next

The outermost <div>:

Carousel require the use of an id for carousel controls to function properly

The class .carousel specifies that this <div> contains carousel

The .slide class adds CSS transition and animation effect, which makes the items slide when showing a new item. Omit this class if you don't want this effect.

The data-ride="carousel" attribute tells Bootstrap to begin animating the carousel immediately whem the page loads.

The indicators part:

The indicators are the little dots at the bottom of each slide(which indicates how many slides there is in the carousel, and which slide the user is currently viewing).

The indicators are specified in an ordered list with class .carousel-indicators.

The data-target attribute points to the id of the carousel.

The data-slede-to attribute specifies which slide to go to, when clicking on the specific dot.

The "Wrapper for slides" part:

The slides are specified in a <div> with class .carousel-inner.

The content of each slide is defined in a <div> with class .item. This can be text or images

The .active class needs to be added to one of the slides. Otherwise, the carousel will not be wisible.

The "Left and right controls" part:

This code adds left and right controll buttons that allows the user to go back and forth between the slides manually"

The data-slide attribute accepts the keywords "prev" or "next", which alerts the slide position relative to its current position.

Add captions to slides

Add <div class="carousel-caption"> within each <div class="item"> to create a caption for each slide
