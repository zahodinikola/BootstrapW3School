Bootstrap dropdowns

Basic dropdown

A dropdown menu is a togglable menu that allows the user to choose one value from a predefined list

The .dropdown class is used to indcate a dropdown menu

Use the .dropdown-menu class to actually build the dropdown menu

To open the dropdown menu use a button or a link with a class of .dropdown-toggle and data-toggle="dropdown"
